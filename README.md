### 注意，本项目仅支持xlsx，不兼容xls

如果是 Mac Retina 屏幕，需要在表格中第一行第四列输入retina

详情见文件 [cmd.xlsx](cmd.xlsx)

存在多个 Sheet 时，仅第一个 Sheet 生效

目前仅提供 Mac Arm64 以及 Windows 64(交叉编译，未经测试) 版本，其他平台需要小伙伴们自行编译。
